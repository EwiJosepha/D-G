#  D&J COLLECTION 
## About
The D&G Collection is a real estate project that allows users to perform CRUD (Create, Read, Update, Delete) operations on property listings. It has both a front-end and back-end component.
## Features 
User authentication (login/registration)
Create new property listings
View a list of all available properties
Update existing property details
Delete property listings
Search and filter properties by various criteria (location, price, size, etc.)

## Technologies Used
 * Front-end:Next.js, CSS, tailwind, JavaScript
 * Back-end: Node.js, Prisma, Nest
 * Database: Postgres
 * Authentication: JSON Web Tokens (JWT)
 * Other tools: Axios,  Bcrypt, etc.

## Installation
1. Clone the repository
git clone https://github.com/your-username/dg-collection.git
2. Install depencies
cd dg-collection
npm install
3. Set up environment variables by creating a .env file at the root of your directory and add the following variables
postgres Url=<your-postgres-connection-string>
JWT_SECRET=<your-jwt-secret-key>(for the back end only)
4. Start the developmemt server 
npm start
